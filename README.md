# thanks-bank
SlackとKtorを用いたSlack連携のサンプルアプリケーションです。主に次の3つを実現します。

- Slackで感謝や応援や励ましの気持ちを伝える
- いつでも、気軽に、みんなでを大事にする
- 送った言葉をWebアプリケーション上で見ることができる
